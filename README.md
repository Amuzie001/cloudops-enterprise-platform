12-Day CloudOps Enterprise Platform Bootcamp (Self Built)

This repository documents my transition from Technical Support Engineer to DevOps Engineer through 12 days of hands-on cloud engineering projects. Each day builds toward a production-style cloud platform while documenting lessons learned, architecture decisions, and implementation details.

# CloudOps Enterprise Platform

A production-style DevOps portfolio built from the ground up to demonstrate practical experience in cloud infrastructure, Infrastructure as Code (IaC), containerization, CI/CD, monitoring, logging, automation, and DevOps best practices using AWS, Azure, Docker, Terraform, GitHub Actions, Traefik, Prometheus, Grafana, and Python.

## Project Objectives

- Provision cloud infrastructure using Terraform
- Automate deployments with GitHub Actions
- Containerize applications using Docker
- Configure reverse proxy with Traefik
- Implement monitoring with Prometheus and Grafana
- Centralize logging using Loki
- Apply Infrastructure as Code (IaC) and DevOps best practices
- Build a production-style cloud platform from scratch
