# CloudOps Enterprise Platform

A production-style DevOps portfolio built from scratch using AWS, Azure, Docker, Terraform, GitHub Actions, Traefik, Prometheus, Grafana, and Python.

## Objectives

- Provision infrastructure using Terraform
- Deploy applications using Docker
- Implement CI/CD with GitHub Actions
- Configure monitoring and logging
- Apply DevOps and Infrastructure as Code best practices

## Technology Stack

- Ubuntu 24.04 (WSL2)
- Git
- Docker
- Docker Compose
- Terraform
- AWS CLI
- Azure CLI
- Python
- GitHub Actions
- Traefik
- Prometheus
- Grafana
- Loki
