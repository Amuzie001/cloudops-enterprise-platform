# Code of Conduct

## My Pledge

I am are committed to creating a welcoming, respectful, and collaborative environment for everyone participating in this project.

## Expected Behavior

- Be respectful and professional.
- Welcome constructive feedback.
- Share knowledge openly.
- Focus on solving problems collaboratively.
- Respect different perspectives and experiences.

## Unacceptable Behavior

- Harassment or discrimination.
- Personal attacks.
- Offensive language.
- Publishing confidential information.
- Any conduct that creates an unprofessional environment.

## Enforcement

Project maintainers reserve the right to remove contributions or participants whose behavior violates this Code of Conduct.

This project follows the spirit of the Contributor Covenant.
